# Breathing Shapes

A guided breathing exercise PWA with beautiful visualizations.

**Live site:** https://igor-breathe.surge.sh

## Features

- 10 different visualizations (Box, Orbit, Blob, Bar, Ladder, Trapezoid, Flower, Minimal, Ring, Path)
- Customizable breathing patterns (Box and Trapezoid types)
- 4-phase breathing cycle: Inhale → Hold → Exhale → Hold
- Haptic feedback and voice prompts
- Progressive Web App (installable, works offline)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Generate PWA icons
npm run icons
```

## Tech Stack

- React 18
- TypeScript
- Vite
- PWA (vite-plugin-pwa)
